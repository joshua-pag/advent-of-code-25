# Advent of Code 2025

Dies ist eine Sammlung von Lösungen zu den Rätseln des Advent of Code 2025. (https://adventofcode.com/)

## Abgeschlossene Rätsel

- Tag 1 ⭐ abgeschlossen am 01.12.2025 - Lösung in C `/day01`

### Anmerkung

Aufgrund der Lizenzbestimmungen des Advent of Code sind die Inputs der Rätsel nicht in diesem Repository enthalten. Um den Code ausführen zu können, ist es also notwendig, zuerst den entsprechenden Input auf der Website (https://adventofcode.com/) zu finden und in eine (oder mehrere) Dateien entsprechend der Dateinamen innerhalb des Codes zu platzieren (meistens `input.txt`).
